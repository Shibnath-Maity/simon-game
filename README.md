# 🎮 Simon Says Game

A simple Simon Says Game built using **HTML, CSS, and JavaScript**.

## 🚀 Features

- Interactive gameplay
- Color animations
- Increasing difficulty
- Game over and restart functionality

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

- ## 🎮 How to Play

1. Press **any key** to start the game.
2. Watch the button that lights up.
3. Click the same colored button.
4. A new color is added each level.
5. Repeat the entire sequence in the correct order.
6. If you click the wrong button, the game is over.
7. Press **any key** to restart and try again.
